# infinitas-skill

Private skill repository for Claude Code / Codex / OpenClaw generated skills.

## Purpose

This repository stores private agent skills, templates, scripts, and references.
It is intended for skills that may contain workflow details, internal notes, or other non-public context.

## Structure

```text
skills/                  Active skills
templates/basic-skill/   Starter template for new skills
docs/conventions.md      Repository conventions
SECURITY.md              Safety rules for private content
```

## Rules

- Keep the repository private.
- Do not commit API keys, tokens, cookies, auth exports, or raw credential files.
- Put reusable skills in `skills/`.
- Start new skills from `templates/basic-skill/`.
